# Brain Tumor Segmentation Using Image Processing

## Overview
This project focuses on detecting and segmenting brain tumors from MRI images using image processing techniques. The system analyzes MRI scans and identifies tumor regions through preprocessing, segmentation, and feature extraction.

## Features
- MRI image preprocessing
- Noise reduction and image enhancement
- Tumor segmentation
- Feature extraction
- Tumor detection and visualization

## Technologies Used
- MATLAB
- Image Processing Toolbox

## Project Files
- Brain Tumor Report (PDF)
- Project Presentation (PPT)

## Applications
- Medical image analysis
- Tumor detection assistance
- Academic research and learning
