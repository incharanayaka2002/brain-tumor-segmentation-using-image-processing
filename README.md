# Brain Tumor Segmentation Using Image Processing

## Project Overview
This project focuses on detecting and segmenting brain tumors from MRI images using image processing techniques in MATLAB. The system enhances MRI scans, identifies tumor regions, and assists in medical image analysis through automated processing.

## Objectives
- Detect brain tumor regions from MRI images
- Perform image preprocessing and enhancement
- Apply segmentation techniques for tumor identification
- Support medical diagnosis through image analysis

## Technologies Used
- MATLAB
- Image Processing Toolbox
- MRI Image Analysis
- Threshold Segmentation
- Morphological Operations

## Methodology
1. MRI Image Input
2. Image Preprocessing
3. Noise Removal using Gaussian Filtering
4. Image Enhancement
5. Threshold Segmentation
6. Morphological Operations
7. Tumor Detection and Visualization

## Output
The system generates:
- Enhanced MRI Image
- Segmented Tumor Region
- Tumor Boundary Detection
- Tumor Visualization Results
